# dialect-orthography 中国方言正字正音库
> 整理中国各地方言的标准用字、权威发音标注与常用词汇，所有内容均经过专业方言词典校验，适配语音标注、NLP训练、方言保护等场景使用。

## ⚠️ 生僻字字体安装说明

本仓库收录大量方言专用字（如𰍧、𫽞、𲚏、𡎚、𤆵、𱟛、𬡊等）及古汉语本字，部分字符位于 Unicode 扩展区 B～F 区，普通系统字体无法显示。**请务必安装以下字体**：

## 📍 标注规范
1. **正字**：优先采用权威方言词典收录的标准汉字，区分本字（古汉语留存）、方言专用字（新造字）、借音字三类
2. **发音**：
   - 拼音：采用各方言区通用拼音方案
   - 国际音标：宽式标音，符合方言语音学规范
   - 调值：采用数字标注对应调类（1=阴平、2=阳平、3=上声、4=去声，多方言调类不同单独说明）
3. **释义**：优先收录方言专属义项，通用义项不重复标注
4. **例词**：选取日常高频使用表达

## 📍 标注规范
1. **正字**：优先采用权威方言词典收录的标准汉字，避免俗写、自造字
2. **发音**：
   - 拼音：采用各方言区通用拼音方案
   - 国际音标：宽式标音，符合方言语音学规范
   - 调值：采用数字标注对应调类（1=阴平、2=阳平、3=上声、4=去声，多方言调类不同单独说明）
3. **释义**：优先收录方言专属义项，通用义项不重复标注
4. **例词**：选取日常高频使用表达
